# Evidence folder

Place all screenshots, scan outputs, log files, and other evidence here.

## Naming Convention

```
ex-NN-description.png        # Screenshot from exercise NN
ex-NN-tool-output.txt        # Tool output / logs
ex-NN-scan-report.html       # HTML scan reports (ZAP, Garak etc.)
```

## What to capture

- Terminal screenshots showing tool output
- Browser screenshots of the web UI
- Scan reports from Garak, ZAP, Bandit, Semgrep
- Log extracts showing attack success or failure

Do not commit large video files — use screenshots instead.
